# Website Traffic Analysis using SQL

This project focuses on analyzing website traffic trends using advanced SQL concepts and Window Functions.

## Concepts Practiced
- LAG() & LEAD()
- Window Functions
- CTEs
- Rolling calculations
- Traffic spike analysis
- Consecutive streak detection
- Percentage growth/drop calculations
- Gaps & Islands pattern

## Business Problems Solved
1. Previous day traffic comparison
2. Day-to-day traffic difference
3. Traffic increase detection
4. Traffic drop greater than 30%
5. Traffic spike greater than 50%
6. Highest traffic spike analysis
7. Rolling average traffic
8. Continuous increasing traffic streaks

## SQL Skills Used
- Window Functions
- ROW_NUMBER()
- Aggregate Functions
- CASE statements
- Analytical SQL patterns
- Time-series analysis

## Key Learning
Learned how analytical SQL can be used to identify trends, spikes, streaks, and behavioral patterns in time-series business data.
