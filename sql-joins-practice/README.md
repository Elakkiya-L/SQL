---

## 🧠 Key Learnings

- JOIN order can affect final output
- LEFT JOIN preserves unmatched rows unless filtered by INNER JOIN later
- GROUP BY defines the level of aggregation
- HAVING filters grouped results (not WHERE)
- COUNT(column) ignores NULL values, COUNT(*) does not
